如何在Mac上安装闪迪安全

4月 5 disk, security, 翻译 评论 字数统计: 606(字) 阅读时长: 2(分)

SanDisk Security 允许设置密码保护和硬件加密，以保护驱动器免受未经授权的使用或访问。

SanDisk Security for Mac 入门：

重要: SanDisk Security 软件使用设置的密码以电子方式锁定和解锁驱动器。如果忘记密码，将无法在其上写入新数据，并且无法访问驱动器中已有的数据。必须先擦除驱动器，然后才能再次使用。

关于闪迪安全软件

SanDisk Security™ 软件实施密码保护和支持的 SanDisk 硬盘驱动器的 256 位硬件数据加密功能。使用闪迪安全软件可以：

  • 创建密码以防止其他人访问驱动器上的文件
  • 更改密码
  • 删除密码保护
  • 解锁任何计算机上的驱动器——无论那里是否安装了闪迪安全软件

操作系统兼容性

SanDisk Security 软件与以下操作系统兼容：

  • High Sierra
  • Sierra
  • El Capitan
  • Yosemite
  • Mavericks

SanDisk Security 软件不支持其他制造商的硬盘驱动器或最初未为 SanDisk Security 软件配置的 SanDisk 硬盘驱动器。

兼容性可能会有所不同，具体取决于硬件配置和操作系统。为获得最佳性能和可靠性，请始终安装最新的更新和服务包 (SP)。转到 Apple 菜单并选择软件更新。

请按照以下步骤安装 SanDisk Security for Mac：

    1. 您可以直接从硬盘安装软件，也可以点击链接从硬盘支持页面下载软件。
    • SanDisk Extreme Portable SSD v2：SanDisk Security Windows 下载
    • SanDisk Extreme Pro 便携式 SSD v2：SanDisk Security Mac 下载
    1. 找到文件并打开 sandisk-security-mac-1-0-0-13.dmg 文件
    1. 双击运行 DMG 文件
    1. 双击 Exec 文件夹
    1. 双击运行闪迪安全安装程序
    1. 单击打开以接受运行从 Internet 下载的文件
    1. 阅读 EULA（最终用户许可协议）并选中复选框以同意 EULA 以启用安装按钮，然后单击安装
    1. 输入计算机的管理用户密码，然后单击确定
    1. 安装完成后点击完成

闪迪安全软件已安装，应用程序文件夹中会显示一个启动程序的图标。请按照以下步骤使用 Finder 打开软件

    1. 打开 Finder 并选择左侧面板上的应用程序
    1. 找到并双击 SanDisk Security 图标以打开
    1. SanDisk Security 应该会启动
